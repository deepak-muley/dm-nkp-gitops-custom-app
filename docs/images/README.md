# Grafana Dashboard Screenshots

This directory contains screenshots of the Grafana dashboard demonstrating the end-to-end working setup.

## Required Images

Please add the following screenshot files to this directory:

1. **grafana-dashboard-request-metrics.png** - Screenshot showing request metrics, performance indicators, and instance-level breakdowns
2. **grafana-dashboard-metrics-overview.png** - Screenshot showing comprehensive metrics overview including request rate, active connections, duration percentiles, and response sizes

## Image Specifications

- **Format**: PNG (recommended) or JPG
- **Recommended size**: 1920x1080 or higher for clarity
- **Naming**: Use the exact filenames listed above to match the README references

## Adding Images

1. Take screenshots of your Grafana dashboard showing the metrics
2. Save them with the exact filenames listed above
3. Place them in this directory (`docs/images/`)
4. Commit and push to the repository

The README.md in the root directory references these images to demonstrate the working end-to-end setup.
